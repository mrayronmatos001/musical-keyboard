## 🌐 How to Contribute
```bash
  # Install the Dependencies by npm
  npm install -g sass
```
```bash
  # Install the Dependencies by yarn
  yarn add sass --dev
```
```bash
  # Clone the Project
  git clone https://github.com/mrayronmatos001/musical-keyboard.git
```
```bash
  # Enter Directory
  cd musical-keyboard
```
